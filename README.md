# vue-dynamic-component2

## 下载项目
```
git clone https://github.com/zhuzhiqiang/vue-dynamic-component2.git
```

## 安装依赖
```
npm install
```

### 启动项目
```
npm run start
```

### 编译组件
```
npm run build:componentA
```
### 打包
```
npm run build
将根目录下的 dist 目录改名为  dynamicComponent2  复制到 tomncat 的 webapps 下面
```
